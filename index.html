Mozilla Monument Name Finder

Want to know where your name is on the fabulous Mozilla monument outside the San Francisco office?

This will show you. Simply search for your name and activate it in the list of names. You'll see a yellow rectangle on the rotating model on this page. Move your mouse on the model to stop it from rotating.

Thanks to each and every one of the 4536 people on this monument, you did make a difference and you still do. Without you, we'll all be using one browser in offices for lots of money and hate the web.

If you want a mobile version that is much lighter and works offline go to the mobile finder.

Written by Chris Heilmann, using a dataset provided by Chris More. Rotating column thingamabob created with 3D CSS Tester. Want to play with this? Go get mozilla-monument-name-finder on GitHub.
